## 🚀 Projects

### [Qzar](https://qzar.in) — Exam & Test Management Platform
> Multi-portal SaaS for coaching institutes — Staff, Student, and Institution portals

- Built with **Next.js 15 App Router**, TypeScript, Prisma ORM, MongoDB, Tailwind CSS
- Secure test-taking engine with **10 anti-cheat measures** (tab switch detection, copy-paste block, fullscreen enforcement)
- Image-supported question bank with GCS signed URLs and WebP conversion
- Role-based auth across three portals (ADMIN / SUPERVISOR / QUESTION_AUTHOR)
- AI question generator using Anthropic API — reduces authoring time from 20 min to 2 min

### [Easycheckin](https://www.easycheckin.co.in) — Hotel Check-in Management System
> Cloud-based SaaS for hotels to manage guest records and check-ins

- Full-stack Next.js app with server components, server actions, and custom APIs
- Integrated **Razorpay** (payments), **Sendgrid** (email), **GCS** (file storage)
- Prisma ORM + MongoDB for database layer
